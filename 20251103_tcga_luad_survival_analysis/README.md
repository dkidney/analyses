# Survival analysis of LUAD data from TGCA

Raw data:

* ~500 samples

* ~20000 genes

Raw expression features:

* gene-level

* RSEM-normalized counts (RNA-Seq by Expectation-Maximization)

* log2(x+1) transformed

Raw survival features:

* event_col: 'OS'

* time_col: 'OS.time'

* censoring rate: ~64%
